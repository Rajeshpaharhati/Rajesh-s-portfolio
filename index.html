Daily Expense Tracker Systen

The Daily Expense Tracker System (DETS) is a robust web application designed to help users efficiently manage their daily expenses. Built using PHP and MySQL, this system provides a comprehensive dashboard where users can view their expenses on a daily, monthly, and yearly basis, offering a clear financial overview. Users can easily add and delete expenses, ensuring up-to-date and accurate records. The system includes a detailed expense report feature, allowing users to analyze their spending patterns over various periods. Additional functionalities include user profile management, password change capabilities, and secure logout options. Furthermore, a password recovery feature ensures users can regain access without hassle. This project showcases my skills in full stack development, focusing on creating user-friendly, secure, and efficient web applications.
